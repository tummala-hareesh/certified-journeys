What is certified-journeys?

A collection of free, open-source certification trackers for senior engineers and data professionals. Every tracker is a single HTML file — no build tools, no frameworks, no sign-ups, no cost.

Each journey gives you a structured day-by-day study plan, progress tracking saved in your browser, practice quizzes, AI-powered study prompts, and exam prep details — all in one place.

Why we built it

  • Free resources exist (docs, YouTube, open assessments) — but nobody built a structured daily plan
  • No existing repo says "here is exactly what to do on Day 1, 2, 3" with progress tracking
  • Avoid spending hours creating learning plans from scratch for every certification
  • Learn in public — if your journey worked, fork it, improve it, share it

How to fork and deploy in 3 steps

  • Fork on GitHub — go to github.com/certified-journeys, find the repo for your cert, click Fork
  • Edit two arrays — open index.html, replace the days and topics arrays with your cert's plan
  • Enable GitHub Pages — Settings → Pages → main branch → Save. Live in 2 minutes

No configuration needed. Your username and repo name are auto-detected from the URL. Your localStorage key is automatically unique to your fork.

Principles

  • Single HTML file — no build tools, no node_modules, no CI needed
  • localStorage only — no server, no account, no tracking
  • MIT licensed — fork, modify, redistribute freely
  • Zero external dependencies except Google Fonts
  • No premium tier, no upsell, no email capture — ever

Contributing

  • Fork a repo and submit a PR with improvements to the study plan
  • Build a new tracker for a cert not yet covered
  • Share your score and learning notes in GitHub Discussions

Hard rules: no tracking, no analytics, no paywalls, no external JS libraries.

Privacy

All progress is stored locally in your browser via localStorage. Nothing is sent to any server. No cookies. No analytics. No tracking pixels. Clearing browser data resets your progress.

Manage journeys

Toggle journeys on or off. Inactive journeys are hidden from your home page but their progress is preserved.
